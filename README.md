## Usage

```
./configure --help
usage: ./configure [options]

options:
  --env=<env>: current env (dev|labo|prod)
  --with-docker: enable docker
  --with-proxy: add proxy to docker
  --clean-before-tasks: remove build files

All invalid options are silently ignored!
```
